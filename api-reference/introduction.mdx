The **KALP Wallet API** provides a full suite of wallet & blockchain utilities:

- Wallet creation (Custodial)
- Native & Token balance
- Token details
- Native transfers
- Token transfers
- Gasless (relayer) transactions
- Contract compilation
- Contract deployment
- Contract write functions
- Transaction status checks
